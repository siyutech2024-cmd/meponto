# MePonto Module Product Specification Template

每个业务模块必须有产品策划书。模块合同用于确认模块能否接入 PontoSys，产品策划书用于指导开发人员具体做什么、怎么做、做到什么程度可以验收。

建议文件路径：

```txt
docs/modules/{module-name}-product-spec.md
```

## 1. 模块基本信息

```txt
模块名称：

开发负责人：
当前状态：idea | planned | beta | active
目标上线阶段：
关联模块合同：
关联路由：
关联 Feature Flag：
```
